# what is TDD?
TDD means Test Driven Developemnt
it is a software develoment process where
you write the test first (it will fail at first)
then you write just enough code to make the test pass
finally you refractor the code for clarity and efficiency
it follow a red----> green----->Refractor Cycle


# What is Selenium ?

In Software: A popular open-source framework used to automate web browsers.
It is widely used by developers and testers to write scripts that simulate human interactions (clicking, typing, navigating) to test web applications.
